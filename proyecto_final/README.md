## Descripción
Proyecto de transformación y limpieza de datos aplicado a una encuesta de preferencias alimentarias. Se aplican técnicas de manipulación, limpieza y análisis exploratorio usando Python, Pandas y NumPy en Google Colab.

## Estructura del proyecto
```
food_preference_project/
├── data/
│   ├── raw/
│   │   └── Food_Preference.csv        # Dataset original sin modificar
│   └── processed/
│       └── food_preference_clean.csv  # Dataset limpio generado por el notebook
├── notebooks/
│   └── proyecto.ipynb                 # Notebook principal (Google Colab)
└── README.md
```

## Cómo reproducir
1. Abrir `notebooks/proyecto.ipynb` en Google Colab
2. Subir `data/raw/Food_Preference.csv` al entorno de Colab
3. Ejecutar las celdas en orden
